# WebCraft — Разработка сайтов на заказ

Одностраничный лендинг для веб-студии с полным набором инструментов для сбора и обработки заявок.

## Функционал

### 🎨 10 блоков
1. **Hero** — заголовок с болью клиента, счётчики (50+ проектов, 4 года, 47 клиентов, 6 стран), таймер обратного отсчёта акции
2. **Услуги** — 12 типов сайтов с ценами и ссылками на кейсы
3. **Процесс** — 7 этапов работы со сроками
4. **Почему мы** — 6 преимуществ с цифрами и доказательствами
5. **Портфолио** — 9 проектов с фильтрацией по типу
6. **Тарифы** — 3 тарифа с выделением популярного
7. **Калькулятор** — интерактивный расчёт с 4 параметрами и 9 опциями
8. **Отзывы** — слайдер с 5 реальными отзывами
9. **FAQ** — 15 частых вопросов (accordion)
10. **Footer** — контакты, соцсети, форма быстрой связи

### 🧮 Калькулятор стоимости
- Тип сайта (8 вариантов)
- Количество страниц (слайдер 1–50)
- Дизайн (стандартный / индивидуальный / премиум)
- Срочность (стандартная / +30% / +60%)
- Функционал (9 чекбоксов)
- Прогресс-бар заполнения
- Анимация изменения цены

### 📋 Многошаговый квиз (7 шагов)
1. Цель сайта
2. Тип сайта
3. Сфера деятельности
4. Срок запуска
5. Бюджет
6. Дополнительные опции
7. Контакты

Прогресс-бар, навигация (назад/далее), экран "Спасибо" с номером заявки и примерной стоимостью.

### 🪟 Модальные окна
- Квиз (по кнопке "Получить расчёт")
- Форма с данными из калькулятора
- Форма с пресетом тарифа
- Обратный звонок
- Exit-intent (скидка 10%)
- Cookie-баннер

### 📊 Сбор данных
Каждая заявка собирает:
- UTM-метки
- Информация об устройстве (userAgent, экран, язык, часовой пояс)
- Поведение на сайте (время, глубина скролла)
- Нумерация заявок (WEB-2026-0001)

### 📬 Отправка
- Email через send.php (сохраняет в `leads.txt` при отсутствии SMTP)
- Telegram Bot API (требуется токен, placeholder: `YOUR_BOT_TOKEN`)
- Live feed — уведомления о новых заявках на сайте

### 🖼️ Технологии
- Чистый HTML5 + CSS3 + Vanilla JS (один файл)
- CSS Custom Properties
- Intersection Observer (scroll-reveal, counter, nav active)
- Адаптив: Mobile (320px), Tablet (768px), Desktop (1280px+)
- Unsplash API для фоновых изображений
